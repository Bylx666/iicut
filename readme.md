# 艾剪

## 介绍

艾剪是一个

1. 开放API
2. 可在线演示视频，可传输工程文件
3. 性能不错
4. 开源免费
5. 纯前端，文件安全
6. 加载完成后可离线使用

的**在线剪辑网站**。

设计参考pr，上手方便。

## 使用

### 浏览器

推荐使用`Chrome`和一切`Chromium`内核的浏览器如`Edge`，本网站兼容至chrome 40，相当于8年前的主流浏览器。

其次可以用`Firefox`，如果火狐遇到卡顿的话

1. 进入`about:config`，将`gfx.canvas.accelerated`设置为`true`
2. 在设置-常规-性能-使用推荐的性能设置**取消勾选**-自动启用硬件加速**不用勾选**
